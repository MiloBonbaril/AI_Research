# Architecture Atlas

3D visualizer for the architectures in `../models/` — one tab per model, each
rendered as a stack of its actual transformer blocks (color/shape-coded by
mixer kind: full attention, local/global attention, GLA, linear attention,
compressed attention, ternary attention). Oneira's world-simulation head and
per-horizon world operators branch off the main stack as separate nodes.

## Run

```bash
bun install
bun run dev      # http://localhost:3000, hot reload
```

## Data

`public/data/models.json` is generated by introspecting the real model
classes (default config, CPU, no forward pass) — not hand-encoded. Regenerate
it after changing anything under `../models/`:

```bash
bun run gen-data   # or: .venv/bin/python webapp/scripts/gen_data.py (from repo root)
```
